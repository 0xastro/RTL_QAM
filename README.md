# RTL_QAM
The project is designed using VHDL to realise the M-QAM modulation.
#### Design Software:  
- GHDL 
- Scansion
- Xilinx Vivado Webpack

#### System Modules
- Phase Accumlator
- Complex Mapper
- DDFS [Sinsoidal Oscillator [sin and cos] ]
- Integrator
- LFSR [Linear Phase Shift Register]
- LATCH [4069]
